# AGENTS.md - Frontend Web (Wakiña Bolivia)

## 🎯 Objetivo

Definir lineamientos para el desarrollo del frontend web de Wakiña
Bolivia, asegurando consistencia, escalabilidad y alineación con la
identidad de marca.

------------------------------------------------------------------------

## 🧩 Alcance

-   Landing page (presentación)
-   Catálogo de productos
-   Integración con WhatsApp
-   Base para futura expansión a e-commerce

------------------------------------------------------------------------

## 🎨 Lineamientos de Diseño

### Estilo visual

-   Minimalista
-   Orgánico
-   Moderno pero cálido

### Colores

-   Verde natural (#4CAF50 o similar)
-   Blanco (#FFFFFF)
-   Tonos tierra (#A67C52, #8D6E63)

### Tipografía

-   Sans-serif moderna (ej: Inter, Poppins)

------------------------------------------------------------------------

## 📱 Responsividad

-   Enfoque mobile-first
-   Breakpoints recomendados:
    -   Mobile: \<768px
    -   Desktop: \>1024px

------------------------------------------------------------------------

## ⚙️ Arquitectura Frontend

### Estructura sugerida

-   /components
-   /pages
-   /assets
-   /styles

### Componentes clave

-   Navbar
-   Hero
-   ProductCard
-   CategorySection
-   Footer
-   WhatsAppButton

------------------------------------------------------------------------

## 🛒 Catálogo de Productos

### Estructura de producto

-   nombre
-   imagen
-   precio (opcional)
-   categoría

### Categorías principales

-   Frutas
-   Verduras
-   Combos familiares

------------------------------------------------------------------------

## 📲 Integración WhatsApp

Formato de enlace:
https://wa.me/`<numero>`{=html}?text=`<mensaje>`{=html}

Ejemplo: "Hola, quiero pedir el combo familiar"

------------------------------------------------------------------------

## 🚀 Buenas prácticas

-   Optimización de imágenes (WebP recomendado)
-   Lazy loading
-   Código modular y reutilizable
-   Evitar dependencias innecesarias
-   SEO básico (meta tags)

------------------------------------------------------------------------

## 🔮 Escalabilidad

Preparar el frontend para: - Integración con backend - Sistema de
carrito - Gestión de usuarios

------------------------------------------------------------------------

## ✅ Resumen

El frontend debe transmitir: - Frescura - Confianza - Cercanía

Y permitir una experiencia simple, rápida y clara para el usuario.
